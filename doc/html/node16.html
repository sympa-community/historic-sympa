15.4 Sharing data with other applications

You may extract subscribers, owners and editors for a list from any of :

  • a text file

  • a Relational database

  • a LDAP directory

See user_data_source list parameter #user-data-source#1965>, page~#user-data-source#1966>.

The three tables can have more fields than the one used by Sympa, by defining these additional fields, they will be available from within Sympa's authorization scenarios and templates (see #db-additional-subscriber-fields#1967>, page~#db-additional-subscriber-fields#1968> and #db-additional-user-fields#1969>, page~#db-additional-user-fields#1970>).

See data inclusion file #data-inclusion-file#1971>, page~#data-inclusion-file#1972>.


15.5 Subscriber count

The number of subscribers of a list can be get from an external application by requesting function 'subscriber_count' on the Web interface.

Example: http://my.server/wws/subscriber_count/mylist
