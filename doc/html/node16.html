15.4 Sharing data with other applications

You may extract subscribers, owners and editors for a list from any of :

  • a text file

  • a Relational database

  • a LDAP directory

See user_data_source list parameter #user-data-source#1994>, page~#user-data-source#1995>.

The three tables can have more fields than the one used by Sympa, by defining these additional fields, they will be available from within Sympa's authorization scenarios and templates (see #db-additional-subscriber-fields#1996>, page~#db-additional-subscriber-fields#1997> and #db-additional-user-fields#1998>, page~#db-additional-user-fields#1999>).

See data inclusion file #data-inclusion-file#2000>, page~#data-inclusion-file#2001>.


15.5 Subscriber count

The number of subscribers of a list can be get from an external application by requesting function 'subscriber_count' on the Web interface.

Example: http://my.server/wws/subscriber_count/mylist
