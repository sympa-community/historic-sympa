Configuration files

  • /usr/local/sympa-ff//etc/sympa.conf
    The main configuration file. See 7, page .

  • /usr/local/sympa-ff//etc/wwsympa.conf
    WWSympa configuration file. See 1.2, page .

  • edit_list.conf
    Defines which parameters/files are editable by owners. See 18.4.4, page .

  • topics.conf
    Contains the declarations of your site's topics (classification in WWSympa), along with their titles. A sample is provided in the sample/ directory of the sympa distribution. See 16.5, page .

  • auth.conf
    Defines authentication backend organisation ( LDAP-based authentication, CAS-based authentication and sympa internal )

  • robot.conf
    It is a subset of sympa.conf defining a Virtual robot (one per Virtual robot).
